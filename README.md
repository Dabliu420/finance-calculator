# Financial Calculator (Python)

A command-line financial calculator written in Python that allows users to perform basic financial computations.

## Description

This program provides three main functionalities:

* Bond pricing (with or without coupons)
* Duration calculation for a bond or a financial portfolio
* Monthly mortgage payment calculation

The user can choose the desired operation from a menu and input the required financial parameters.

## How to Run

Make sure Python is installed, then run:

```python
python your_file_name.py
```

## Features

* Bond price calculation using discounted cash flows 
* Portfolio duration computation
* Mortgage payment and total interest calculation
* Interactive command-line interface

## Technologies

* Python
* Standard library (math, basic I/O)

## Notes

This project is intended for educational purposes and demonstrates fundamental financial concepts and Python programming techniques.
